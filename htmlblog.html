After Writing this much Save the file name as index.html . Make a Folder named Blog and save the index.html file in that folder .

You need a Browser so that you can view your demo blog on your Computer . If you are on a Windows / Linux Machine then these are the Best Choices :-

  • Google Chrome
  • Mozilla Firefox
If you are on a Mac Machine then Safari is the Best Choice . If you are on Android then the inbuilt Chrome Browser is enough for you along with Mi File Manager.

[For Computer]Open the File Manager on your Computer and then open the Blog Folder . Then right click on the index.html file and open it in your Browser .
[For Android] Open the Mi File Manager and Search for the Blog Folder . Open the folder and click on the check box of index.html file and then tap on more and then tap Details . From there , Copy the Location of the file which will be in blue colour . Open the Chrome Browser and paste the location in the Search Box and then click Search .
So now you have your Blog on the Browser . Remember that this Blog is stored on your Computer and not on the internet .

So now that you have Setup your Environment , we can start doing the real job which is Coding .
